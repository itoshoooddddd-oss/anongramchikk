# 📁 Файлы проекта Anongram

## Основные файлы

### 🔹 Backend (Python Flask)
- **app.py** - Главный файл приложения (488 строк кода)
  - Маршруты и обработчики
  - Модели базы данных
  - Генерация seed-фраз
  - API endpoints

### 🔹 Конфигурация
- **requirements.txt** - Python зависимости
- **Procfile** - Команда запуска для продакшена
- **runtime.txt** - Версия Python (3.11.0)
- **.gitignore** - Исключения для Git

### 🔹 Документация
- **README.md** - Полная документация (EN)
- **DEPLOYMENT_GUIDE.md** - Инструкция по публикации
- **START_RU.md** - Быстрый старт (RU)
- **PROJECT_SUMMARY.md** - Этот файл

## Frontend

### HTML Шаблоны (templates/)
1. **auth_choice.html** - Главная страница (выбор вход/регистрация)
2. **register.html** - Страница регистрации
3. **login.html** - Страница входа
4. **messenger.html** - Основной интерфейс мессенджера

### CSS Стили (static/css/)
- **style.css** - Все стили приложения (722 строки)
  - Адаптивный дизайн для ПК и мобильных
  - Градиентные элементы
  - Анимации
  - Темная тема

### JavaScript (static/js/)
1. **auth.js** - Логика регистрации и входа (103 строки)
   - Обработка форм
   - Копирование seed-фраз
   - Автозаполнение из буфера обмена
   
2. **messenger.js** - Логика мессенджера (291 строка)
   - Управление чатами
   - Отправка сообщений
   - Создание групп/каналов
   - Добавление контактов

## База данных

Автоматически создается при первом запуске:
- **anongram.db** - SQLite база данных

Таблицы:
- Users (пользователи)
- Contacts (контакты)
- Groups (группы/каналы)
- GroupMember (участники групп)
- Messages (сообщения)

## Запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python app.py

# Доступ
http://localhost:5000
```

## Функционал

### ✅ Реализовано

1. **Аутентификация**
   - Выбор вход/регистрация
   - Регистрация: логин → ник → 13 seed-фраз
   - Вход по логину + seed-фразам
   - Автозаполнение seed-фраз из clipboard

2. **Пользователи**
   - Уникальные логины
   - Уникальные никнеймы
   - Генерация 13 seed-фраз

3. **Мессенджер**
   - Личные сообщения
   - Группы
   - Каналы
   - Контакты

4. **UI/UX**
   - Профессиональный дизайн
   - Адаптивность (ПК + мобильные)
   - Градиенты и анимации
   - Темная тема

## Технические характеристики

- **Backend**: Python 3.11 + Flask 3.0.0
- **Database**: SQLite + SQLAlchemy 3.1.1
- **Frontend**: HTML5 + CSS3 + Vanilla JavaScript
- **Server**: Gunicorn 21.2.0 (для продакшена)
- **Migration**: Flask-Migrate 4.0.5

## Безопасность
