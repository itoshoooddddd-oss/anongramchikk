# 🚀 Быстрый старт - Anongram Messenger

## Запуск локально (на вашем компьютере)

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Запуск приложения
```bash
python app.py
```

### 3. Открыть в браузере
Перейдите по адресу: **http://localhost:5000**

## Как пользоваться

### Регистрация нового пользователя
1. На главной странице нажмите **"Register"**
2. Введите уникальный **Login** (для входа в систему)
3. Введите уникальный **Nickname** (будет виден другим пользователям)
4. Система сгенерирует **13 seed-фраз** - СОХРАНИТЕ ИХ!
5. Скопируйте seed-фразы или запишите
6. Нажмите **"Proceed to Login"**

### Вход в аккаунт
1. На главной странице нажмите **"Login"**
2. Введите ваш **Login**
3. Введите ваши **13 seed-фраз** (можно вставить из буфера обмена)
4. Нажмите **"Login"**

### Использование мессенджера

#### Добавить контакт
1. Нажмите кнопку **"+ Add Contact"**
2. Введите nickname пользователя
3. Нажмите **"Add Contact"**

#### Создать группу
1. Нажмите кнопку **"👥 New Group"**
2. Введите название группы
3. Нажмите **"Create Group"**

#### Создать канал
1. Нажмите кнопку **"📢 New Channel"**
2. Введите название канала
3. Нажмите **"Create Channel"**

#### Отправить сообщение
1. Выберите контакт, группу или канал в списке слева
2. Введите текст сообщения в поле ввода
3. Нажмите **"📤 Send"** или клавишу Enter

## Публикация в интернете

### ⚠️ Важно о my.canva.site

**my.canva.site поддерживает только статические сайты.** Поскольку Anongram использует Python Flask (серверная часть), вы НЕ МОЖЕТЕ разместить его на my.canva.site напрямую.

### ✅ Рекомендуемые альтернативы (БЕСПЛАТНО):

#### 1. Render.com (Лучший вариант)
1. Зарегистрируйтесь на https://render.com
2. Создайте новый "Web Service"
3. Подключите GitHub репозиторий или загрузите файлы
4. Деплой автоматический!

Подробные инструкции см. в файле **DEPLOYMENT_GUIDE.md**

#### 2. Railway.app
1. Зарегистрируйтесь на https://railway.app
2. Создайте новый проект
3. Подключите GitHub
4. Автоматический деплой

#### 3. PythonAnywhere
1. Зарегистрируйтесь на https://www.pythonanywhere.com
2. Загрузите файлы через файловый менеджер
3. Создайте Flask приложение
4. Настройте и запустите

## Решение проблем

### Ошибка базы данных
Удалите файл `anongram.db` и перезапустите приложение.

### Порт уже занят
Измените порт в файле `app.py` (строка 488).

### Не работает на my.canva.site
Используйте Render.com вместо my.canva.site (см. выше).
